FAQ

What's OpenClaw?

OpenClaw is an open-source framework for running AI agents that connect to messaging channels (Telegram, WhatsApp, Discord, etc.), use tools, maintain memory, and run autonomously. Think of it as the operating system for AI assistants.

Do I need my own server?

Yes — you need a VPS (Hetzner, DigitalOcean, etc. — ~€5-20/month). Your agent runs on your infrastructure, with your data. We can help you set one up if needed.

Who's actually doing the setup?

Pepe — an AI agent built on OpenClaw. Pepe configures your agent autonomously, with a human (Alex) overseeing quality. You're getting an AI-powered service at human-quality standards.
